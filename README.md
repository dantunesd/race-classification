# Race Classification

The purpose of this application is classify a race.

---

## Pre-requirements

`docker 17.06+`

`docker-compose 1.22+`

---

## Overview

The application was developed with Golang 1.12.

---

## Run

```bash
make run-app
```

The classification file `result.csv` will be found at `data` folder

---

## Tests

Run unit tests:

```bash
make run-unit-tests
```
